👨‍💻 About The Project

Built by Moksedul Ali

Infinity Hub started as a simple idea: create a personal technology dashboard using only a mobile phone and free online tools.

The project was developed by Moksedul Ali with the assistance of ChatGPT. From creating the GitHub repository to publishing the website online, every step was completed through learning, experimentation and continuous improvement.

The goal of Infinity Hub is to bring Search, AI, Knowledge and Social Platforms together in one futuristic and easy-to-use interface.
